[GLOBAL CANON FOR THIS STAGE]

Новый clean rebuild строится вокруг:

GLOBAL IMPULSE GRAPH.

Базовый объект:

IMPULSE.

Импульс хранит:

- TF;
- direction;
- start;
- end;
- high;
- low;
- range;
- previous reference;
- current ownership state;
- next candidate state.

Главное структурное событие:

следующий импульс снимает HIGH или LOW предыдущего импульса.

Слом НЕ определяется:

- 0.79;
- OTE;
- глубиной коррекции;
- произвольным protected level;
- одиночной внутренней линией;
- визуальной меткой.

Fib 0/1 = high/low импульса.

Fib 0.5–0.79 = correction / OTE / quality / execution layer.

Профили только A и B.

PROFILE A:
- POI TF = 4H / 240
- PRIMARY TF = 1H / 60
- CONFIRM TF = 15m / 15

PROFILE B:
- POI TF = 1D / D
- PRIMARY TF = 4H / 240
- CONFIRM TF = 1H / 60

DISPLAY TF = экран.
PROFILE TF = логика.
